\subsection{Logistic Regression Model}
The overall results of the Logistic Regression model could have been improved. 
For example, for the second variation of the model, I believe with actually fine-tuning DistilBERT rather than using it to obtain static features, the model could have yielded better results. 
Yet, the improvement from the base model to the second variation is notable, unlike the decrease in performance from the base to the first variation. 
Overall, to an individual approaching this problem, I would recommend obtaining a balanced dataset and attempting different data augmentation techniques to allow the model to properly learn aspects of the different genres. 
This way the model can better correlate words to its context and recognize patterns more efficiently thereby yielding more accurate results for its predictions.
